## 📌 About This System

Supplyverify provides a way to track items through a supply chain using digital records. This system uses blockchain technology to create a permanent history for each product. Manufacturers, retailers, and customers use this tool to confirm the origin and movement of goods.

The system uses three main components. First, it uses a smart contract on the Ethereum network to store data. Second, it uses a web interface for daily tasks. Third, it uses QR codes to link physical products to their digital history. Users scan these codes to view product information instantly.

## 🔍 How to Use the Verification Tool

The verification tool helps you confirm that a product is genuine. Each product has a unique QR code attached to its packaging.

1.  Open the application on your computer.
2.  Navigate to the Verify Product section.
3.  Use your device camera or a scanner to read the physical QR code.
4.  The system pulls the record from the blockchain.
5.  Compare the details on the screen with the physical item.

The blockchain provides an unchangeable record of each step the product took. If the data on the screen matches the product in your hand, the item is authentic.

## 🛠️ Managing Your Supply Chain Data

Manufacturers gain access to tools that add items to the ledger. You must have the necessary permissions within the smart contract to perform these actions.

1.  Navigate to the Manufacturer Dashboard.
2.  Enter the product details, such as the name, serial number, and batch information.
3.  Click Submit to sign the transaction.
4.  Confirm the transaction in your MetaMask wallet.
5.  The system generates a unique QR code. Print this code and attach it to your physical product.

The process ensures that every item receives a digital identity that stays with it during transit.

## ❓ Frequently Asked Questions

**Does this cost money to use?**
This system uses the Sepolia test network. This network uses test tokens that have no monetary value. You do not pay real money to use the system.

**What happens if the website is down?**
The core data lives on the blockchain. Even if the website interface is slow, your registered product data remains safe and public. You can always use other blockchain explorer websites to view your product history.

**Can I delete an entry?**
Blockchain records are permanent. You can add new state updates to an item, but you cannot delete the history of an entry once it appears on the ledger. This creates the trust required for a valid supply chain.

**Why does the page ask for a signature?**
The application requires a digital signature to prove you own the wallet account. It does not transfer funds or give away your identity. It simply authorizes you to view or change data regarding your products.
